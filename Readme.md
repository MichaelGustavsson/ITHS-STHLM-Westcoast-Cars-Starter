# Westcoast Cars

## Detta arkiv kommer innehålla min kod för Westcoast Cars applikationen

Applikationen kommer att innehåll projekt för ett REST baserat API samt flera olika _frontend_ projekt.

Minst ett för kunder som vill kunna lista bilar och boka provkörning samt minst två projekt för säljare på Westcoast Cars.

Ett projekt som är byggt med ASP.NET MVC eller ASP.NET Razor pages samt en Single Page Applikation(SPA).
